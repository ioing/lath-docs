MainScrollId

AppletConfig > mainScrollId 指定 Applet 的主滚动容器，其与 [tapStatusBarToScrollToTop] 和 [pullToRefresh] 能够联合使用。

Types

 interface AppletBaseConfig {
              mainScrollId?: string
            }

Details

如果没有指定主滚动容器，则默认 <define-applet> 元素为主滚动。

Example

 createApplication({
              applets: {
                home: {
                  config: {
                    mainScrollId: 'root'
                  }
                }
              }
            })

Notes

注意指定的滚动容器“id”不可存在于 ShadowDom 中。
